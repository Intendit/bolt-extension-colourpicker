### Colourpicker Extension for Bolt

Here's an example of how to use Colourpicker extension as a field in contenttypes.yml file:

```
banner:
    type: colourpicker
    label: "Choose A Banner Colour"
    values:
        "#000": Solid Black
        "#E1E1E1": Almost White
        "#444": Dark Grey
        "#FF0011": Bright Red
```
